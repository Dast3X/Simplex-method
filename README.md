# Simplex algorithm
To use simplex
<ul>
  <li>put .txt file in 'Problem' folder</li>
  <li>launch the program 'Simplex method.exe'</li>
  <li>write the name of the file in the console including .txt
	e.g 'problem.txt'</li>
  <li>solution will be saved in the folders Decimal & Fraction</li>
</ul>
